# Social Media Analyser

**RPA Solution (UiPath, GenAI)**

Social Media Analyser is an automation project designed to help influencers monitor their social media presence. The RPA robot collects posts and comments from social media accounts on TikTok, calculates engagement metrics, evaluates audience sentiment using GenAI and generates consolidated reports in Excel. Automated email notifications are sent for posts exceeding predefined engagement thresholds.

---

## Features

- **Automated Data Collection:** Gather posts and comments from social media accounts for influencers.
- **Engagement Metrics:** Calculate likes, comments and overall engagement per post.
- **Sentiment Analysis:** Evaluate audience feedback (positive/negative/neutral) using GenAI on the first comment.
- **Excel Reporting:** Centralize all collected data and metrics into an organized Excel report.
- **Email Notifications:** Automatically send alerts for posts that exceed engagement thresholds.
- **End-to-End RPA Automation:** Fully automated workflow from data collection to report generation and notifications.
